# E-commerce Sales Analysis

This project analyzes e-commerce sales data using SQL to understand revenue trends and customer behavior.

## Business Question
How are sales performing over time, and which customers and products drive the most revenue?

## Dataset
- E-commerce sales data  
- One row represents one order  
- Includes information about orders, customers, and revenue

## Approach
1. Explore the data structure
2. Clean and validate records
3. Analyze sales and customer metrics using SQL
4. Extract business insights
